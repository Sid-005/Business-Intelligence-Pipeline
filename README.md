README: Running instructions for COMP3134

1. Navigate to project directory
2. Run the following command to install dependencies
```
   pip install -r requirements.txt
   pip3 install -r requirements.txt
```

4. Once the dependencies are installed, run the program with the following instruction
   ```
   python run initial.py
   python3 run initial.py
   ```


Alternatively,
1. View the results of the code in output folder
   - Visualizations: Contains all the generated graphs of the analysis, clustering and bundling
   - Cleaned-Data: Contains all the cleaned datasets used to generate graphs and business insights
  
2. Read the analysis summary .txt file to learm the results of the analysis
